Exercise details are described in [PROBLEM.md](./PROBLEM.md)

## Installation

```bash
go build
./quiz
```

## Usage

```bash
Usage of quiz:
  -file string
        the path to file with questions (default "problems.csv")
  -limit int
        the timer for answering questions in seconds (default 30)
  -shuffle
        shuffle questions
```
